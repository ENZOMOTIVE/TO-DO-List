# TO DO List

`TO-DO-List` is a static web project using HTML, CSS, and JavaScript. It can be opened directly in a browser or served locally for development.

## Features

- Static browser application that can be opened locally

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

- `README.md` - project documentation
- `index.html` - static app entry page
- `index.js` - browser JavaScript
- `style.css` - CSS styles

## Getting Started

### Prerequisites

- A modern web browser

### Setup

```bash
git clone https://github.com/ENZOMOTIVE/TO-DO-List.git
cd TO-DO-List
```

Open `index.html` in a browser, or serve the folder with a lightweight static server.
